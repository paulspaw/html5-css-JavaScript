## css 基础语法 🌰：MyCss.css
>```
> selector {
>    property: value
> }
>```
> 属性大于1个之后，属性之间用分号隔开，
> 如果值大于一个单词，则需要加上引号：
>```css
> eg. h1 {color:red;font-size:14px;}
> p {font-fmaily:"sans serif";}
>```
